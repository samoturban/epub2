EPUB → PDF (automaticky přes GitHub Actions)

Důležité: Pokud jsi nahrál tento repozitář přes web, je potřeba ještě jednou vytvořit workflow soubor na GitHubu: .github/workflows/convert_epub.yml (viz README).

Jak použít:

  1. Nahraj své .epub soubory do složky /epub.
  2. GitHub Actions je převede do PDF a uloží do /pdf.
  3. Seznam PDF dole se obnoví po doběhnutí workflow (~1–2 min).
